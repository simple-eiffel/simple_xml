Overview

SIMPLE_XML_QUICK provides one-liner XML parse and query operations with XPath support. Perfect for config files and simple XML processing.

For XML building and modification, use SIMPLE_XML directly.

Quick Start

local
    xml: SIMPLE_XML_QUICK
    titles: ARRAYED_LIST [STRING]
do
    create xml.make

    -- XPath query - get all matching text content
    titles := xml.xpath (book_xml, "//book/title")

    -- Get first match
    if attached xml.first (book_xml, "//book/title") as title then
        print (title)
    end

    -- Get attribute value
    if attached xml.attr (book_xml, "//book", "id") as id then
        print ("Book ID: " + id)
    end

    -- Simple element access (no XPath)
    if attached xml.text (html, "title") as t then
        print ("Page title: " + t)
    end

    -- Get all element texts
    across xml.texts (html, "p") as p loop
        print (p)  -- all paragraph texts
    end

    -- Count matching nodes
    print ("Books: " + xml.count (library_xml, "//book").out)

    -- Check if exists
    if xml.exists (config_xml, "//database/host") then ...

    -- Build simple elements
    print (xml.element ("name", "Alice"))
    -- <name>Alice</name>

    -- Validation
    if xml.is_valid (some_xml) then ...
end

API Reference

Parsing

FeatureDescription
parse (xml)Parse XML string to document
parse_file (path)Parse XML file to document

XPath Queries

FeatureDescription
xpath (xml, query)Get text content of all matches
first (xml, query)Get text content of first match
attr (xml, query, name)Get attribute from first match
count (xml, query)Count matching nodes
exists (xml, query)Does at least one match exist?

Simple Element Access

FeatureDescription
text (xml, element)Get text by tag name (first match)
texts (xml, element)Get all text by tag name

Building

FeatureDescription
element (name, content)Create simple element
element_with_attrs (name, attrs, content)Create element with attributes

Validation

FeatureDescription
is_valid (xml)Is string valid XML?
has_errorDid last operation fail?
last_errorLast error message
